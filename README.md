# FSML
A machine learning project written in F#

### Version
0.1.2

### Algorithms implemented:
  - Linear Regression
    * no penalty
    * L1 penalty via coordinate descent
    * L2 penalty
  - Logistic Regression
    * no penalty
    * L1 penalty via coordinate descent
    * L2 penalty via iterated reweighted least square
  - SVM via Sequential Minimal Optimization (SMO)
    * linear kernel
    * rbf(Gaussian) kernel
